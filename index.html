About

I am an Engineer II at Danovo Energy Solutions and a software engineering researcher interested in empirical software engineering, LLM agents in software development, automated code review, pull-request analysis, software maintenance, and applied data science for power-system modeling and long-term resource planning.

Empirical Software Engineering, LLM Agents & Applied Data Science

My work connects practical software engineering with research questions about how modern AI-assisted development changes code review, maintenance, and pull-request behavior. I am especially interested in measuring how agentic software development tools perform in real engineering workflows and where their risks appear during maintenance.

In industry, I build software and data systems for power-system analysis, including FastAPI services, TypeScript dashboards, automated simulation workflows, GIS analysis, Oracle APEX applications, and power-flow modeling for utility and data-center interconnection planning.
